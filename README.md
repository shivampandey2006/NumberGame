# NumberGame



A simple number guessing game made using **HTML + JavaScript**.  
The game generates a **random number between 1 and 10**, and the user has to guess it.
 Features
- Random number generated on every reload
- Shows hint:
  - Too High
  - Too Low
  - Correct Guess!
- Fully works offline
- Super beginner-friendly
Technologies Used
- HTML
- JavaScript
 How to Run
1. Download or copy the project code.
2. Save file as: `game.html`
3. Open the file in any browser (Chrome, Firefox, Edge)
4. Start guessing the number!
